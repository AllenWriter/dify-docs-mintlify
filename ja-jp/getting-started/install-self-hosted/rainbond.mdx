title: Rainbondでのデプロイ方法
---

### Dify のデプロイ
1. [AI アプリマーケット](https://hub.grapps.cn)にアクセスします。
2. トップページの精選アプリの中で `Dify` を探し、「インストール」をクリックします。
3. Rainbond のアカウントを登録した後、直接インストールとデプロイを行います。
4. トポロジー図の状態が緑になるまで待ち、それはデプロイが成功したことを示します。大体 `1-3` 分かかります。

### Dify へのアクセス
Rinbond が提供するサブドメインを使用することができます。また、カスタムドメインを紐付けることも選択できます。

ページの青色の「アクセス」ボタンを直接クリックしてアクセスします。
