### 部署 Dify
1. 访问[AI 应用市场](https://hub.grapps.cn)
2. 在首页精选应用中找到 `Dify`，点击`安装`
3. 注册 Rainbond 账号之后直接安装部署
4. 等待拓扑图状态变绿代表部署成功，大概需要`1-3`分钟。

### 访问 Dify

你可以使用 Rinbond 提供的子域名，也可以选择绑定自定义域名。

直接点击页面蓝色 `访问` 按钮进行访问
