---
title: 工具简介
---

工具可以帮助用户可以扩展 LLM 的能力，在 Dify 平台上创建更强大的 AI 应用，例如新增联网搜索、科学技术或图片绘制能力。例如你可以为 Agent 类型应用添加合适的工具，它可以通过任务推理、步骤拆解、调用工具完成更加复杂的任务。

## 工具类型

Dify 支持以下四种工具类型：

1. **插件（Plugins）** - 对开发者更加友好，可扩展性更强的第三方服务扩展模块。详细说明请参考[插件简介](/zh-hans/plugins/introduction)。
2. **自定义工具（Custom Tools）** - 用户可以导入自定义的 API 工具（OpenAPI 格式）。开发说明请参考[快速接入工具](/zh-hans/guides/tools/quick-tool-integration)。
3. **Workflow as Tool** - 将 Dify 内构建好的工作流应用转为工具并在应用中调用。
4. **MCP 服务器（HTTP）** - 支持 HTTP 协议的 MCP 服务器。

## 配置插件

参考以下文档，了解如何配置插件。


### 使用工具

目前，你可以在"工作室"中创建**智能助手型应用**时，将已配置好凭据的工具在其中使用。

![](https://assets-docs.dify.ai/2025/04/70c01de59cce8fce27b62040ab729e78.png)

以下图为例，在财务分析应用内添加工具后，智能助手将在需要时自主调用工具，从工具中查询财务报告数据，并将数据分析后完成与用户之间的对话。

![](https://assets-docs.dify.ai/2025/04/cbeb1e1435fa56c43c3be957aac1c5f2.png)

### 管理工具

在 Dify 的工具页面，您可以：

- 查看和管理所有类型的工具
- 使用"All Tags"按钮筛选不同类型的工具
- 使用搜索功能快速定位特定工具
- 查看工具的详细信息和使用说明
